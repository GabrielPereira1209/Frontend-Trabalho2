Projeto frontend estático de exemplo (VagasJá)

Requisitos:
- Node.js (v16+ recomendado) e npm

Instalação e execução local rápida:

1. Instale dependências dev:

```bash
npm install
```

2. Compile TypeScript e sirva site estático:

```bash
npm run start
```

Isso compila `scripts/main.ts` para `scripts/main.js` e serve o diretório na porta `5173`.

Se preferir apenas abrir `index.html`, abra no navegador o arquivo localmente.
